# Character Roster

**Phase 1 — Launch + Expansion**

The first 5–10 characters cover completely different audiences. Characters are the UI for Aura's second brain — each one maps personality to features.

**Launch lineup:** Mochi, Pixel, Sakura, Nova, Ember

Platform overview: [Character Platform](../features/character-platform.md)

---

## Category 1: Cute Animals 🐾

Highest retention. People keep these running all day.

### Mochi — Shiba Inu Puppy 🐕

**Launch character.** Default companion for new users.

| Attribute | Detail |
|-----------|--------|
| Audience | Everyone — broadest appeal |
| Vibe | Loyal, enthusiastic, always happy to see you |
| Primary features | Quick Notes, Tasks |

**Behavior:**

- Sleeps on window borders
- Chases mouse cursor
- Wags tail when tasks completed
- Brings notes in mouth

**Interactions:**

| Level | Action |
|-------|--------|
| Single click | Bark + tail wag |
| Double click | Open Quick Notes |
| Double click (alt) | Fetch today's tasks |
| Feed treat | Happy animation + celebrate |
| Long press | Fetch note from timeline |

**Special trick:** Runs across all monitors chasing the cursor.

**Spawn objects:** Sticky Note, Task Card
