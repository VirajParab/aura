# Contributing to AuraOS

Thank you for your interest in AuraOS. This project is open source under the [GNU Affero General Public License v3.0](LICENSE).

## Before you contribute

1. Read the [Contributor License Agreement (CLA)](CLA.md). **By submitting a pull request, you agree to the CLA.**
2. Check [DEVELOPMENT.md](DEVELOPMENT.md) for setup (`make dev`, `make check`).
3. Open an issue for large changes before starting work.

## Pull request process

1. Fork the repository and create a branch from `main`.
2. Keep changes focused; match existing code style and conventions.
3. Run `make check` before opening a PR.
4. In your PR description, note what changed and how to test it.
5. Confirm CLA acceptance (see [CLA.md](CLA.md)).

## What we welcome

- Bug fixes and tests
- Character engine, overlay, and platform improvements
- Documentation corrections
- Community character packs (see `characters/`)

## Premium features

Some product areas are reserved for commercial / hosted offerings and may not accept community implementations without prior discussion. See [Open source vs premium](README.md#open-source-vs-premium) in the README.

## Code of conduct

Be respectful and constructive. Maintainers may close or reject contributions that do not align with project direction or license requirements.
